# 🌍 Earth Bridge Alliance

An [Astro](https://astro.build) starter project with [Tailwind CSS v4](https://tailwindcss.com) integration.

## 🚀 Getting Started

### Prerequisites

- Node.js v18 or higher

### Install Dependencies

```bash
npm install
```

### Development

Start the local development server:

```bash
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

### Build

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## 🛠️ Tech Stack

| Tool | Description |
